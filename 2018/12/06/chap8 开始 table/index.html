小知识点

prototype cell: a cell template can be reused.

Protocol(协议)

Delegation pattern: 一个对象依赖于另一个对象执行特定任务。

打个比方，你开公司，请了一位顶尖的图像设计师，你必须提供足够的信息给他，他才能根据要求设计你的 logo，而你太忙了，所以派个人助理将信息提供给他。

现实生活 iOS 作用
图像设计师 UITableView 显示数据
你的个人助理 delegate object(委托对象) 提供问题的答案
protocols 提各种问题

UITableView: 我可以显示数据，但你必须提供给我信息。

Protocols: 提问题，UITableView 需要什么信息，例如

UITableViewDataSource: 数据和 Table View 之间的连接，定义了必须采用的方法

  • 表格有几部分？
  • 几行？
  • 你想在第 x 行显示啥内容？
  • 各个部分的标题是什么？
  • …

UITableViewDelegate: 外观

  • 行高？
  • 头部分的高度？
  • 第 x 行的锁进级别是多少？
  • …

Delegate object: 遵守协议，提供实现，即回答问题。ViewController

重复利用 cell

当处理成千上万行数据时，如果我们为每一个数据都申请一个 cell，内存爆炸。因此我们需要从队列回收带有特定标识符，可重复使用的 table cell。

为什么可以回收？

屏幕大小是有限的，即便总共有 1000 条数据要显示，一屏幕最多显示 10 条。

如何回收？

回顾数据结构所学的知识。有一个队列，当一个 cell 划出屏幕时，入队列；滑进屏幕待使用的 cell 出队列。
