状态栏颜色

因为详细页图片颜色较深，我们最好将状态栏换成白色来获取更好的视觉效果。在 RestaurantDetailViewController 类中 override preferredStatusBarStyle属性。

然而，状态栏依旧没有改变，因为 detail view controller 是嵌入在 navigation controller 中，iOS 仍然会使用 navigation controller 的状态栏风格，换句话说，就是 navigation 把 detail view 覆盖住了。

如何设置呢？有一个叫 childViewControllerForStatusBarStyle的属性，它能设定哪一个 controller 用来决定状态栏风格，默认为 navigation controller，所以，我们要 override 它。
