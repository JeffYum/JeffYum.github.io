Python Note(2)

Posted on 2019-03-08 | Edited on 2019-04-06 | In Notes |

这本教材讨论了巨量的函数。留个印象，以后要用到的时候知道有这么个函数。本文讨论数字和字符串函数。

数字

数学函数

  • abs()
  • ceil() 天花板
  • floor() 地板
  • max(), min()
  • round(x,n), 保留小数点后几位，n 默认为 0

import math

  • exp() need import math, math.exp(x)
  • fabs(), 也是绝对值，不过只支持整数和浮点数
  • log(), 自然对数。log10()
  • modf(), 返回一个数的小数和整数部分，元组，两部分的符号和原数相同
  • pow(x,y), xy
  • sqrt(), 平方根

随机函数

随机数可应用于游戏，模拟，测试，安全，隐私等方面。

  • choice(seq), 从列表、元组、字符串中随机返回一项

import random

  • seed([x],[y]), 初始随机数，在使用任何随机函数之前调用。

    时间 seed(),整数 seed(10), 字符串 seed("hello", 2)

  • randrage(start,stop,step), 范围内随机

  • random(), 返回 0 到 1 之间的浮点数

  • shuffle(list), 洗牌

  • uniform(x,y), 随机 x 到 y 之间的浮点数，包含 x，不包含 y

三角函数

  • degrees(), 弧度转角度
  • radians(), 角度转弧度

import math

  • acos(x), arc cosine. (-1,1). asin()
  • atan()
  • atan2(y,x), atan(y / x)
  • cos(), sin(), tan()
  • hypot(x,y), Euclidean norm, (x, y) 到原点的距离

数学常数

  • pi
  • e

字符串

用 index 访问字符串的值，切片可以获取子串。var[1:5] 获取第 1 到 4 个元素。

更新字符串，先用切片获取字符串的某一片段，再 + 上需要添加的元素

有一些字符有特定的意义，需要用 \ 来输出它的原始值，查表 Page 100

与 C 语言类似, String Formatting Operator, 注意后面的格式

字符串函数

大小写

  • capitalize(), 开头大写，并返回
  • lower(), 拷贝一份全小写；upper()
  • swapcase(), 反转大小写
  • title(), 拷贝，首字母大写

检查

  • isalnum(), 检查文字和数字是否连体，’this2019’ => True
  • isalpha(), 检查是否纯字母，空格也不行
  • islower(), 纯小写; isupper(), 大写
  • isdigit(), True: Unicode数字，byte数字，全角数字，罗马数字; False: 汉字
  • isnumeric(), True: Unicode 数字，全角数字，罗马，汉字; Error: byte 数字
  • isdecimal(), True: Unicode 数字，全角数字; False: 罗马，汉字; Error: byte 数字
  • isspace(), 纯空格
  • istitle(), 是否所有首字母大写，很形象，标题
  • endswith(suffix, start, end), 范围内是否以特定字符串结尾`
  • startswith(str, beg, end), 以 str 开头

填充，删除，替换

  • center(width, 'fillchar'), 总长，填充

  • ljust(width, fillchar), 左边顶格，右边填充；rjust(width, fillchar)

  • zfill(width), 左侧填 0

  • lstrip([chars]), 拷贝一份，从左侧删除指定字符，默认是空格；rstrip()；strip() 左右

  • replace(old, new, max), 用 new 替换 old，最大替换几个，不给就全部替换

  • expandtabs(tabsize), 拷贝字符串，将制表符 \t 扩展到指定长度

  • maketrans(intab, outtab), 用 outtab 中对应的值替换 intab；translate()

搜索

  • max(), 最大字母；min(), 符号(,.)最小
  • count(sub, start = 0, end = len(string)), 搜索子串在指定 index 范围内出现的次数
  • find(str, beg = 0, end = len(string)) 返回字串首次出现在范围内的 index，否则返回 -1。index(), 没找到报错
  • rfind(str, beg, end), 最后一次被找到的 index 位置，否则 -1。rindex(), 失败报错

其他

  • decode(encoding, errors) and encode(encoding, errors), 解码编码
  • str.join(sequence), 用 str 内容连接序列
  • len(), 长度
  • split(str, num = string.count(str)), 返回列表，用 str 将原串分割成几行，可以指定 num。str 默认是空格，全部分割。splitlines(), 用 \n 分割
