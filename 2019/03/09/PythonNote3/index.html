Python Note(3)

Posted on 2019-03-09 | In Notes |

讨论列表，元组，字典的用法。

list

删除元素 del list[2]

与字符串操作十分类似，长度，加，复读，检查是否存在，迭代

list[-1] 表示最后一个元素，往前走 -2，-3…

内建方法

  • len(list)
  • max(list); min()
  • list(seq), 元组或字符串转换成 list

操作

  • list.append(obj), 添加

  • .count(obj), 统计出现几次

  • .extend(seq), 添加多个

  • .index(onj), 对象第一次出现的 index，否则报错

  • .insert(index, obj), 指定位置插入

  • .pop(index), index 可选，默认弹出最后一个，返回弹出的元素

  • .remove(obj), 移除，不返回任何值

  • .reverse(), 反转

  • .sort([func]), 排序，可使用比较函数 func

Tuples

空元组 tup1 = ()

即使元组中只有一个值，也要在后面加个逗号 tup1 = (50,)

元组的值不可改变，但是可以用存在的元组创建新元组

没有任何括号的默认是元组

除了会对列表做更改的函数，其他函数完全和列表一样。

Dictionary

键值以冒号分隔，元素之间逗号，整个花括号，空字典 {}。

一个字典中键是唯一的，而值不是。值可以是任何类型，键只能是不可更改的类型如字符串，数字，元组。

访问，更新，添加，删除

键的性质

  1. 多个相同的键是不允许的，访问时会使用最后相同的最后一个键。
  2. 键应该是不可更改的类型，用列表来做键就不行

内建方法

  • len(dict)
  • str(dict), 转成字符串
  • type(parameter), 返回参数的类型
  • .copy(), 复制
  • .fromkeys(seq, value), 以 seq 中的元素为键，创建字典，value 可选，作为所有键的值
  • .get(key, default=None), 返回对应键的值，如果没有返回 default(可以自定义); .setdefault(key, default=None), 功能一样，区别是没有就创建
  • .items(), 返回由双元素元组组成的列表
  • .keys(), 列表形式返回所有键
  • .values()，列表形式返回所有值
  • .update(dict2), 将 dict2 的键值对加入
