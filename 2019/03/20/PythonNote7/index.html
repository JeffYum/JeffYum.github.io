异常处理

通常会在函数的开头检查输入，结尾检查输出。

Assert

assert 语句，期望判断结果为 True，否则报错

Try-except-else

Exception 是一个表示错误的对象，当遇到异常时，要么立即处理，要么终止执行。

如果我们有一些可疑的代码，可以用 try: 把他们包起来，保护程序。

接着用个 except:，尽可能优雅地做些处理。一个 try 可以跟多个 except，处理不同种类的异常；也可以多个异常执行相同的处理；也可以提供通用的语句，处理任何异常，不推荐这样做，因为无法定位错误的根源。

最后 else: 块，如果尝试后没有任何异常，则执行这里的语句。

except 可以有参数，下面例子中 Argument 被赋值 ‘xyz’

try-finally

finally: 其中包含的语句必定会执行，即便 try 成功了。

触发异常

Exception 是一个比较基础的 built-in 类。All built-in, non-system-exiting exceptions are derived from this class. All user-defined exceptions should also be derived from this class. 比如 NameError 等。

自定义异常
