Python Note(8)

Posted on 2019-03-20 | In Notes |

Python 的面向对象，之前 iOS 的文章中有更基础详细的讲解。

可以添加，移除，修改属性

替代直接操作，下面有几个访问属性的方法：（为什么要用方法？）

内建类属性

  • __dict__, 包含命名空间的字典
  • __doc__, 说明书
  • __name__, 类名
  • __module__, 定义该类的模块名，交互模式（当前）叫 __main__
  • __bases__, 包含基类的元组

垃圾回收

继承

一个字类可以有多个父类

issubclass(sub, sup) 检查是否是字类

isinstance(obj, Class) 检查是否是类的实例

方法重写

子类中可以重写父类的方法，直接定义相同名称的方法即可，无需加关键字 override

重写基本函数

一些带下划线的通用函数也可以重写，例如前面我们无意中重写了 __init__ 和 __del__， 在里面加入了打印的语句。

运算符重载

假设我们创建了一个表示二维向量的类，需要一个向量中的元素分别相加的功能，常规的 + 号是不能实现的，此时我们可以重新定义 + 号，称为运算符重载。

数据隐藏

让属性不能被类以外的空间看见，双下划线前缀（强私有）。实际上 Python 只是改了下文件名来实现隐藏，我们仍然可以通过形如 object._className__attrName 的格式，在外部访问隐藏属性。

私有属性，私有方法。单下划线表示私有，子类还是允许访问的。在第一篇笔记的开头提到过。
